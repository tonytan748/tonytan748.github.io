Fill Character in String In Python

2015-08-28
分类： python

In this part we will talk about Python String module.


import string

print string.center("abc",5,"-")

# -abc-

print string.ljust("abc",5,"-")

#abc--

print string.rjust("abc",5,"-")

#--abc

print string.zfill("abc",5)

#default fill by "0"
#00abc
