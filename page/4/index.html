Cleaning the data

The next step is to clean the data. I decided to take the classification for granted and simply remove question marks. Also, use one token for each type and change the alternative spellings accordingly. Finally remove superfluous whitespace and start all words with capital letter.

def cleanup_type(s):
    if not isinstance(s, str):
            return s
    s = s.replace('?', '').replace('  ', ' ')
    s = s.replace('volcanoes', 'volcano')
    s = s.replace('volcanoe', 'Volcano')
    s = s.replace('cones', 'cone')
    s = s.replace('Calderas', 'Caldera')
    return s.strip().title()

Next I define a function that will plot a volcano map for the given parameters. Lists of longitudes, latitudes and elevations, that all need to have the same lengths, are mandatory, the other parameters have defaults set.

As mentioned above, sizes correspond to elevations, i. e. a higher volcano will be plotted as a larger triangle. To achieve this the 1st line in the plot_map function creates an array of bins and the 2nd line maps the individual elevation values to these bins.

Next a Basemap object is created, coastlines and boundaries will be drawn and continents filled in the given color. Then the volcanoes are plotted. The 3rd parameter of the plot method is set to ^r, the circumflex stands for triangle and r for red. For more details, see the documentation for plot.

The Basemap object will be returned so it can be manipulated after the function finishes and before the map is plotted, youвҖҷll see why in a later example.

def plot_map(lons, lats, elevations, projection='mill', llcrnrlat=-80, urcrnrlat=90, llcrnrlon=-180, urcrnrlon=180, resolution='i', min_marker_size=2):
    bins = np.linspace(0, elevations.max(), 10)
    marker_sizes = np.digitize(elevations, bins) + min_marker_size

    m = Basemap(projection=projection, llcrnrlat=llcrnrlat, urcrnrlat=urcrnrlat, llcrnrlon=llcrnrlon, urcrnrlon=urcrnrlon, resolution=resolution)
    m.drawcoastlines()
    m.drawmapboundary()
    m.fillcontinents(color = '#333333')

    for lon, lat, msize in zip(lons, lats, marker_sizes):
        x, y = m(lon, lat)
        m.plot(x, y, '^r', markersize=msize, alpha=.7)

    return m

2.3.2 жҸҗдҫӣйқҷжҖҒж–Үд»¶

еҪ“зј–еҶҷWebеә”з”Ёж—¶пјҢдҪ жҖ»еёҢжңӣжҸҗдҫӣеғҸж ·ејҸиЎЁгҖҒJavaScriptж–Үд»¶е’ҢеӣҫеғҸиҝҷж ·дёҚйңҖиҰҒдёәжҜҸдёӘж–Үд»¶зј–еҶҷзӢ¬з«ӢеӨ„зҗҶеҮҪж•°зҡ„вҖқйқҷжҖҒеҶ…е®№вҖқгҖӮTornadoжҸҗдҫӣдәҶеҮ дёӘжңүз”Ёзҡ„жҚ·еҫ„жқҘдҪҝе…¶еҸҳеҫ—е®№жҳ“гҖӮ

2.3.2.1 и®ҫзҪ®йқҷжҖҒи·Ҝеҫ„

дҪ еҸҜд»ҘйҖҡиҝҮеҗ‘Applicationзұ»зҡ„жһ„йҖ еҮҪж•°дј йҖ’дёҖдёӘеҗҚдёәstatic_pathзҡ„еҸӮж•°жқҘе‘ҠиҜүTornadoд»Һж–Үд»¶зі»з»ҹзҡ„дёҖдёӘзү№е®ҡдҪҚзҪ®жҸҗдҫӣйқҷжҖҒж–Үд»¶гҖӮAlpha Mungerдёӯзҡ„зӣёе…ід»Јз ҒзүҮж®өеҰӮдёӢпјҡ

еңЁиҝҷйҮҢпјҢжҲ‘д»¬и®ҫзҪ®дәҶдёҖдёӘеҪ“еүҚеә”з”Ёзӣ®еҪ•дёӢеҗҚдёәstaticзҡ„еӯҗзӣ®еҪ•дҪңдёәstatic_pathзҡ„еҸӮж•°гҖӮзҺ°еңЁеә”з”Ёе°Ҷд»ҘиҜ»еҸ–staticзӣ®еҪ•дёӢзҡ„filename.extжқҘе“Қеә”иҜёеҰӮ/static/filename.extзҡ„иҜ·жұӮпјҢе№¶еңЁе“Қеә”зҡ„дё»дҪ“дёӯиҝ”еӣһгҖӮ

2.3.2.2 дҪҝз”Ёstatic_urlз”ҹжҲҗйқҷжҖҒURL

TornadoжЁЎжқҝжЁЎеқ—жҸҗдҫӣдәҶдёҖдёӘеҸ«дҪңstatic_urlзҡ„еҮҪж•°жқҘз”ҹжҲҗstaticзӣ®еҪ•дёӢж–Үд»¶зҡ„URLгҖӮи®©жҲ‘д»¬жқҘзңӢзңӢеңЁindex.htmlдёӯstatic_urlзҡ„и°ғз”Ёзҡ„зӨәдҫӢд»Јз Ғпјҡ

иҝҷдёӘеҜ№static_urlзҡ„и°ғз”Ёз”ҹжҲҗдәҶURLзҡ„еҖјпјҢе№¶жёІжҹ“иҫ“еҮәзұ»дјјдёӢйқўзҡ„д»Јз Ғпјҡ

йӮЈд№Ҳдёәд»Җд№ҲдҪҝз”Ёstatic_urlиҖҢдёҚжҳҜеңЁдҪ зҡ„жЁЎжқҝдёӯзЎ¬зј–з Ғе‘ўпјҹжңүеҰӮдёӢеҮ дёӘеҺҹеӣ гҖӮе…¶дёҖпјҢstatic_urlеҮҪж•°еҲӣе»әдәҶдёҖдёӘеҹәдәҺж–Үд»¶еҶ…е®№зҡ„hashеҖјпјҢе№¶е°Ҷе…¶ж·»еҠ еҲ°URLжң«е°ҫпјҲжҹҘиҜўеӯ—з¬ҰдёІзҡ„еҸӮж•°vпјүгҖӮиҝҷдёӘhashеҖјзЎ®дҝқжөҸи§ҲеҷЁжҖ»жҳҜеҠ иҪҪдёҖдёӘж–Үд»¶зҡ„жңҖж–°зүҲиҖҢдёҚжҳҜд№ӢеүҚзҡ„зј“еӯҳзүҲжң¬гҖӮж— и®әжҳҜеңЁдҪ еә”з”Ёзҡ„ејҖеҸ‘йҳ¶ж®өпјҢиҝҳжҳҜеңЁйғЁзҪІеҲ°з”ҹдә§зҺҜеўғдҪҝз”Ёж—¶пјҢйғҪйқһеёёжңүз”ЁпјҢеӣ дёәдҪ зҡ„з”ЁжҲ·дёҚеҝ…еҶҚдёәдәҶзңӢеҲ°дҪ зҡ„йқҷжҖҒеҶ…е®№иҖҢжё…йҷӨжөҸи§ҲеҷЁзј“еӯҳдәҶгҖӮ

еҸҰдёҖдёӘеҘҪеӨ„жҳҜдҪ еҸҜд»Ҙж”№еҸҳдҪ еә”з”ЁURLзҡ„з»“жһ„пјҢиҖҢдёҚйңҖиҰҒж”№еҸҳжЁЎжқҝдёӯзҡ„д»Јз ҒгҖӮдҫӢеҰӮпјҢдҪ еҸҜд»Ҙй…ҚзҪ®Tornadoе“Қеә”жқҘиҮӘеғҸи·Ҝеҫ„/s/filename.extзҡ„иҜ·жұӮж—¶жҸҗдҫӣйқҷжҖҒеҶ…е®№пјҢиҖҢдёҚжҳҜй»ҳи®Өзҡ„/staticи·Ҝеҫ„гҖӮеҰӮжһңдҪ дҪҝз”Ёstatic_urlиҖҢдёҚжҳҜзЎ¬зј–з Ғзҡ„иҜқпјҢдҪ зҡ„д»Јз ҒдёҚйңҖиҰҒж”№еҸҳгҖӮжҜ”еҰӮиҜҙпјҢдҪ жғіжҠҠйқҷжҖҒиө„жәҗд»ҺжҲ‘д»¬еҲҡжүҚдҪҝз”Ёзҡ„/staticзӣ®еҪ•з§»еҲ°ж–°зҡ„/sзӣ®еҪ•гҖӮдҪ еҸҜд»Ҙз®ҖеҚ•ең°ж”№еҸҳйқҷжҖҒи·Ҝеҫ„з”ұstaticеҸҳдёәsпјҢз„¶еҗҺжҜҸдёӘдҪҝз”Ёstatic_urlеҢ…иЈ№зҡ„еј•з”ЁйғҪдјҡиў«иҮӘеҠЁжӣҙж–°гҖӮеҰӮжһңдҪ еңЁжҜҸдёӘеј•з”ЁйқҷжҖҒиө„жәҗзҡ„ж–Үд»¶дёӯзЎ¬зј–з ҒйқҷжҖҒи·Ҝеҫ„йғЁеҲҶпјҢдҪ е°ҶдёҚеҫ—дёҚжүӢеҠЁдҝ®ж”№жҜҸдёӘжЁЎжқҝгҖӮ

2.3.3 жЁЎжқҝзҡ„дёӢдёҖжӯҘ

еҲ°зӣ®еүҚдёәжӯўпјҢдҪ е·Із»ҸиғҪеӨҹеӨ„зҗҶTornadoжЁЎжқҝзі»з»ҹзҡ„з®ҖеҚ•еҠҹиғҪдәҶгҖӮеҜ№дәҺеғҸAlpha Mungerиҝҷж ·з®ҖеҚ•зҡ„Webеә”з”ЁиҖҢиЁҖпјҢеҹәзЎҖзҡ„еҠҹиғҪеҜ№дҪ иҖҢиЁҖи¶іеӨҹз”ЁдәҶгҖӮдҪҶжҳҜжҲ‘д»¬еңЁжЁЎжқҝйғЁеҲҶзҡ„еӯҰд№ е№¶жІЎжңүз»“жқҹгҖӮTornadoеңЁеқ—е’ҢжЁЎеқ—зҡ„еҪўејҸдёҠд»Қз„¶жңүдёҖдәӣжҠҖе·§пјҢиҝҷдёӨдёӘеҠҹиғҪдҪҝеҫ—зј–еҶҷе’Ңз»ҙжҠӨеӨҚжқӮзҡ„Webеә”з”ЁжӣҙеҠ з®ҖеҚ•гҖӮжҲ‘д»¬е°ҶеңЁз¬¬дёүз« дёӯзңӢеҲ°иҝҷдәӣеҠҹиғҪгҖӮ

жҲ‘д»¬д№ӢеүҚдҪҝз”ЁRequestHandlerеҜ№иұЎзҡ„get_argumentж–№жі•жқҘжҚ•иҺ·иҜ·жұӮжҹҘиҜўеӯ—з¬ҰдёІзҡ„зҡ„еҸӮж•°гҖӮеҗҢж ·пјҢжҲ‘д»¬д№ҹеҸҜд»ҘдҪҝз”ЁзӣёеҗҢзҡ„ж–№жі•жқҘиҺ·еҫ—POSTиҜ·жұӮдј йҖ’зҡ„еҸӮж•°гҖӮпјҲTornadoеҸҜд»Ҙи§ЈжһҗURLencodedе’Ңmultipartз»“жһ„зҡ„POSTиҜ·жұӮпјүгҖӮдёҖж—ҰжҲ‘д»¬д»ҺPOSTдёӯиҺ·еҫ—дәҶж–Үжң¬е’Ңе®ҪеәҰзҡ„еҸӮж•°пјҢжҲ‘д»¬дҪҝз”ЁPythonеҶ…е»әзҡ„textwrapжЁЎеқ—жқҘд»ҘжҢҮе®ҡзҡ„е®ҪеәҰиЈ…йҘ°ж–Үжң¬пјҢе№¶е°Ҷз»“жһңеӯ—з¬ҰдёІеҶҷеӣһеҲ°HTTPе“Қеә”дёӯгҖӮ

1.2.3 е…ідәҺRequestHandlerзҡ„жӣҙеӨҡзҹҘиҜҶ

еҲ°зӣ®еүҚдёәжӯўпјҢжҲ‘д»¬е·Із»ҸдәҶи§ЈдәҶRequestHandlerеҜ№иұЎзҡ„еҹәзЎҖпјҡеҰӮдҪ•д»ҺдёҖдёӘдј е…Ҙзҡ„HTTPиҜ·жұӮдёӯиҺ·еҫ—дҝЎжҒҜпјҲдҪҝз”Ёget_argumentе’Ңдј е…ҘеҲ°getе’Ңpostзҡ„еҸӮж•°пјүд»ҘеҸҠеҶҷHTTPе“Қеә”пјҲдҪҝз”Ёwriteж–№жі•пјүгҖӮйҷӨжӯӨд№ӢеӨ–пјҢиҝҳжңүеҫҲеӨҡйңҖиҰҒеӯҰд№ зҡ„пјҢжҲ‘д»¬е°ҶеңЁжҺҘдёӢжқҘзҡ„з« иҠӮдёӯиҝӣиЎҢи®Іи§ЈгҖӮеҗҢж—¶пјҢиҝҳжңүдёҖдәӣе…ідәҺRequestHandlerе’ҢTornadoеҰӮдҪ•дҪҝз”Ёе®ғзҡ„еҸӘжҳҜйңҖиҰҒи®°дҪҸгҖӮ

1.2.3.1 HTTPж–№жі•

жҲӘжӯўеҲ°зӣ®еүҚи®Ёи®әзҡ„дҫӢеӯҗпјҢжҜҸдёӘRequestHandlerзұ»йғҪеҸӘе®ҡд№үдәҶдёҖдёӘHTTPж–№жі•зҡ„иЎҢдёәгҖӮдҪҶжҳҜпјҢеңЁеҗҢдёҖдёӘеӨ„зҗҶеҮҪж•°дёӯе®ҡд№үеӨҡдёӘж–№жі•жҳҜеҸҜиғҪзҡ„пјҢе№¶дё”жҳҜжңүз”Ёзҡ„гҖӮжҠҠжҰӮеҝөзӣёе…ізҡ„еҠҹиғҪз»‘е®ҡеҲ°еҗҢдёҖдёӘзұ»жҳҜдёҖдёӘеҫҲеҘҪзҡ„ж–№жі•гҖӮжҜ”еҰӮпјҢдҪ еҸҜиғҪдјҡзј–еҶҷдёҖдёӘеӨ„зҗҶеҮҪж•°жқҘеӨ„зҗҶж•°жҚ®еә“дёӯжҹҗдёӘзү№е®ҡIDзҡ„еҜ№иұЎпјҢж—ўдҪҝз”ЁGETж–№жі•пјҢд№ҹдҪҝз”ЁPOSTж–№жі•гҖӮжғіиұЎGETж–№жі•жқҘиҝ”еӣһиҝҷдёӘйғЁд»¶зҡ„дҝЎжҒҜпјҢиҖҢPOSTж–№жі•еңЁж•°жҚ®еә“дёӯеҜ№иҝҷдёӘIDзҡ„йғЁд»¶иҝӣиЎҢж”№еҸҳпјҡ

жҲ‘д»¬еҲ°зӣ®еүҚдёәжӯўеҸӘжҳҜз”ЁдәҶGETе’ҢPOSTж–№жі•пјҢдҪҶTornadoж”ҜжҢҒд»»дҪ•еҗҲжі•зҡ„HTTPиҜ·жұӮпјҲGETгҖҒPOSTгҖҒPUTгҖҒDELETEгҖҒHEADгҖҒOPTIONSпјүгҖӮдҪ еҸҜд»Ҙйқһеёёе®№жҳ“ең°е®ҡд№үдёҠиҝ°д»»дёҖз§Қж–№жі•зҡ„иЎҢдёәпјҢеҸӘйңҖиҰҒеңЁRequestHandlerзұ»дёӯдҪҝз”ЁеҗҢеҗҚзҡ„ж–№жі•гҖӮдёӢйқўжҳҜеҸҰдёҖдёӘжғіиұЎзҡ„дҫӢеӯҗпјҢеңЁиҝҷдёӘдҫӢеӯҗдёӯй’ҲеҜ№зү№е®ҡfrob IDзҡ„HEADиҜ·жұӮеҸӘж №жҚ®frobжҳҜеҗҰеӯҳеңЁз»ҷеҮәдҝЎжҒҜпјҢиҖҢGETж–№жі•иҝ”еӣһж•ҙдёӘеҜ№иұЎпјҡ

1.2.3.2 HTTPзҠ¶жҖҒз Ғ

д»ҺдёҠйқўзҡ„д»Јз ҒеҸҜд»ҘзңӢеҮәпјҢдҪ еҸҜд»ҘдҪҝз”ЁRequestHandlerзұ»зҡ„ser_status()ж–№жі•жҳҫејҸең°и®ҫзҪ®HTTPзҠ¶жҖҒз ҒгҖӮз„¶иҖҢпјҢдҪ йңҖиҰҒи®°дҪҸеңЁжҹҗдәӣжғ…еҶөдёӢпјҢTornadoдјҡиҮӘеҠЁең°и®ҫзҪ®HTTPзҠ¶жҖҒз ҒгҖӮдёӢйқўжҳҜдёҖдёӘеёёз”Ёжғ…еҶөзҡ„зәІиҰҒпјҡ

TornadoдјҡеңЁHTTPиҜ·жұӮзҡ„и·Ҝеҫ„ж— жі•еҢ№й…Қд»»дҪ•RequestHandlerзұ»зӣёеҜ№еә”зҡ„жЁЎејҸж—¶иҝ”еӣһ404пјҲNot Foundпјүе“Қеә”з ҒгҖӮ

еҰӮжһңдҪ и°ғз”ЁдәҶдёҖдёӘжІЎжңүй»ҳи®ӨеҖјзҡ„get_argumentеҮҪж•°пјҢе№¶дё”жІЎжңүеҸ‘зҺ°з»ҷе®ҡеҗҚз§°зҡ„еҸӮж•°пјҢTornadoе°ҶиҮӘеҠЁиҝ”еӣһдёҖдёӘ400пјҲBad Requestпјүе“Қеә”з ҒгҖӮ

еҰӮжһңдј е…Ҙзҡ„иҜ·жұӮдҪҝз”ЁдәҶRequestHandlerдёӯжІЎжңүе®ҡд№үзҡ„HTTPж–№жі•пјҲжҜ”еҰӮпјҢдёҖдёӘPOSTиҜ·жұӮпјҢдҪҶжҳҜеӨ„зҗҶеҮҪж•°дёӯеҸӘжңүе®ҡд№үдәҶgetж–№жі•пјүпјҢTornadoе°Ҷиҝ”еӣһдёҖдёӘ405пјҲMethos Not Allowedпјүе“Қеә”з ҒгҖӮ

еҪ“зЁӢеәҸйҒҮеҲ°д»»дҪ•дёҚиғҪи®©е…¶йҖҖеҮәзҡ„й”ҷиҜҜж—¶пјҢTornadoе°Ҷиҝ”еӣһ500пјҲInternal Server Errorпјүе“Қеә”з ҒгҖӮдҪ д»Јз Ғдёӯд»»дҪ•жІЎжңүжҚ•иҺ·зҡ„ејӮеёёд№ҹдјҡеҜјиҮҙ500е“Қеә”з ҒгҖӮ

еҰӮжһңе“Қеә”жҲҗеҠҹпјҢе№¶дё”жІЎжңүе…¶д»–иҝ”еӣһз Ғиў«и®ҫзҪ®пјҢTornadoе°Ҷй»ҳи®Өиҝ”еӣһдёҖдёӘ200пјҲOKпјүе“Қеә”з ҒгҖӮ

еҪ“дёҠиҝ°д»»дҪ•дёҖз§Қй”ҷиҜҜеҸ‘з”ҹж—¶пјҢTornadoе°Ҷй»ҳи®Өеҗ‘е®ўжҲ·з«ҜеҸ‘йҖҒдёҖдёӘеҢ…еҗ«зҠ¶жҖҒз Ғе’Ңй”ҷиҜҜдҝЎжҒҜзҡ„з®ҖзҹӯзүҮж®өгҖӮеҰӮжһңдҪ жғідҪҝз”ЁиҮӘе·ұзҡ„ж–№жі•д»Јжӣҝй»ҳи®Өзҡ„й”ҷиҜҜе“Қеә”пјҢдҪ еҸҜд»ҘйҮҚеҶҷwrite_errorж–№жі•еңЁдҪ зҡ„RequestHandlerзұ»дёӯгҖӮжҜ”еҰӮпјҢд»Јз Ғжё…еҚ•1-3жҳҜhello.pyзӨәдҫӢж·»еҠ дәҶеёёи§„зҡ„й”ҷиҜҜж¶ҲжҒҜзҡ„зүҲжң¬гҖӮ

*д»Јз Ғжё…еҚ•1-3 еёёи§„й”ҷиҜҜе“Қеә”пјҡhello-errors.py

еҪ“жҲ‘д»¬е°қиҜ•дёҖдёӘPOSTиҜ·жұӮж—¶пјҢдјҡеҫ—еҲ°дёӢйқўзҡ„е“Қеә”гҖӮдёҖиҲ¬жқҘиҜҙпјҢжҲ‘д»¬еә”иҜҘеҫ—еҲ°Tornadoй»ҳи®Өзҡ„й”ҷиҜҜе“Қеә”пјҢдҪҶеӣ дёәжҲ‘д»¬иҰҶеҶҷдәҶwrite_errorпјҢжҲ‘д»¬дјҡеҫ—еҲ°дёҚдёҖж ·зҡ„дёңиҘҝпјҡ
