еҲ©з”ЁPythonзҡ„еӯ—з¬ҰдёІеӨ„зҗҶжЁЎеқ—пјҢејҖеҸ‘дәәе‘ҳеҸҜд»Ҙзј–еҶҷи„ҡжң¬з”ЁжқҘз”ҹжҲҗйӮЈдәӣж јејҸзӣёеҗҢзҡ„CгҖҒC++гҖҒJAVAжәҗзЁӢеәҸгҖҒеӨҙж–Үд»¶е’ҢжөӢиҜ•ж–Үд»¶пјҢд»ҺиҖҢйҒҝе…ҚеӨ§йҮҸзҡ„йҮҚеӨҚе·ҘдҪңгҖӮжң¬ж–ҮжҰӮиҝ°дёӨз§ҚеҲ©з”ЁPython stringзұ»з”ҹжҲҗjavaжәҗд»Јз Ғзҡ„ж–№жі•гҖӮ

String Template

TemplateжҳҜдёҖдёӘеҘҪдёңиҘҝпјҢеҸҜд»Ҙе°Ҷеӯ—з¬ҰдёІзҡ„ж јејҸеӣәе®ҡдёӢжқҘпјҢйҮҚеӨҚеҲ©з”ЁгҖӮTemplateд№ҹеҸҜд»Ҙи®©ејҖеҸ‘дәәе‘ҳеҸҜд»ҘеҲҶеҲ«иҖғиҷ‘еӯ—з¬ҰдёІзҡ„ж јејҸе’Ңе…¶еҶ…е®№дәҶпјҢж— еҪўдёӯеҮҸиҪ»дәҶејҖеҸ‘дәәе‘ҳзҡ„еҺӢеҠӣгҖӮTemplateеұһдәҺstringдёӯзҡ„дёҖдёӘзұ»пјҢжңүдёӨдёӘйҮҚиҰҒзҡ„ж–№жі•пјҡsubstituteе’Ңsafe_substituteгҖӮжӣҝжҚўж—¶дҪҝз”Ёsubstitute()пјҢиӢҘжңӘиғҪжҸҗдҫӣжЁЎжқҝжүҖйңҖзҡ„е…ЁйғЁеҸӮж•°еҖјж—¶дјҡеҸ‘з”ҹејӮеёёгҖӮдҪҝз”Ёsafe_substitute() еҲҷдјҡжӣҝжҚўеӯҳеңЁзҡ„еӯ—е…ёеҖј,дҝқз•ҷжңӘеӯҳеңЁзҡ„жӣҝжҚўз¬ҰеҸ·гҖӮиҰҒдҪҝз”Ёзҡ„иҜқеҸҜз”Ёд»ҘдёӢж–№ејҸи°ғз”Ёпјҡ

from string import Template

TemplateжңүдёӘзү№ж®Ҡж ҮзӨәз¬Ұ$,е®ғе…·жңүд»ҘдёӢзҡ„и§„еҲҷпјҡ

пјҲ1пјүдё»иҰҒе®һзҺ°ж–№ејҸдёә$xxx,е…¶дёӯxxxжҳҜж»Ўи¶іpythonе‘ҪеҗҚи§„еҲҷзҡ„еӯ—з¬ҰдёІпјҢеҚідёҚиғҪд»Ҙж•°еӯ—ејҖеӨҙгҖҒдёҚиғҪдёәе…ій”®еӯ—зӯүпјӣ

пјҲ2пјүеҰӮжһң$xxxйңҖиҰҒе’Ңе…¶д»–еӯ—з¬ҰдёІжҺҘи§Ұж—¶пјҢз”Ё{}е°ҶxxxеҢ…иЈ№иө·жқҘпјӣ

ејҖеҸ‘дәәе‘ҳйҖҡиҝҮзј–еҶҷtemplateж–Үд»¶пјҢе№¶йҖҡиҝҮTemplateж–№жі•еҲӣе»әжЁЎжқҝгҖҒsubstituteж–№жі•жӣҝжҚўеӯ—з¬ҰдёІеҚіеҸҜеҝ«жҚ·зҡ„з”ҹжҲҗжүҖйңҖзҡ„ж–Үд»¶гҖӮзј–еҶҷtemplateж–Үд»¶ж—¶дёҖе®ҡиҰҒжіЁж„ҸвҖң$вҖқзҡ„дҪҝз”ЁпјҢеӣ дёәPythonдјҡе°Ҷд»ҘвҖң$вҖқејҖеӨҙзҡ„еӯ—з¬ҰдёІзҗҶи§ЈжҲҗйңҖиҰҒжӣҝжҚўзҡ„еҸҳйҮҸгҖӮ

replace

str.replace(old,new[,max])

Python replace() ж–№жі•жҠҠеӯ—з¬ҰдёІдёӯзҡ„ oldпјҲж—§еӯ—з¬ҰдёІпјү жӣҝжҚўжҲҗ new(ж–°еӯ—з¬ҰдёІ)пјҢеҰӮжһңжҢҮе®ҡз¬¬дёүдёӘеҸӮж•°maxпјҢеҲҷжӣҝжҚўдёҚи¶…иҝҮ max ж¬ЎгҖӮ
